`ImageCache`는 **Kingfisher의 하이브리드 캐시 시스템**이다.

즉, **메모리 캐시 + 디스크 캐시를 묶어서 하나의 API로 제공하는 캐시 엔진이다.**

- Memory Storage — 빠른 in-memory access (RAM)
- Disk Storage — 장기/지속 캐싱
- Hybrid API — 두 저장소를 함께 제어 & 조회
